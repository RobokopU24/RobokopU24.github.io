ROBOKOP

ROBOKOP helps researchers explore and derive insights from a biomedical knowledge graph by providing a user-friendly interface that supports the exploration of answer subgraphs and associated provenance and publication support.

Automat API Docs

The documentation site for Automat's API provides information on how to access and use the different knowledge sources available through the Automat service

Additional Tools

Aragorn

A Translator ARA that points to the Automat KPs and SmartAPI registry.

ROBOKOP KG

Query the ROBOKOP knowledge graph directly in the Neo4j browser.

ExEmPLAR

Web interface to ExEmplar, which is a generic query tool to support Cypher queries of Neo4j databases, including the ROBOKOP KG.
